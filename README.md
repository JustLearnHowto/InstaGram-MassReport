### INSTALLATION [Termux] (Not working for android right now) 
```bash
* `apt update`
* `apt upgrade`
* `pkg install python`
* `pkg install python3`
* `pkg install git`
* `git clone https://github.com/Crevils/InstaReport`
* `cd instareport`
* `pip install -r requirements.txt`
* `chmod +x *`
* `bash setup.sh` OR `python ReportBot.py`
```
